## Тех. детали

- Браузер: Chrome через chrome-devtools MCP, viewport 1440×900, full-page screenshots
- Виджет Nuxt DevTools в правом нижнем углу был скрыт перед каждым кадром
- Эмуляция состояний (`01`, `04`, `05`, `06`) сделана через `initScript` с подменой `window.fetch` — UI и стейт не трогались, только сетевой ответ
- PDF был сохранён в `/tmp/anya-report.pdf` (17 КБ), затем 1-я страница → PNG через `magick -density 144 file.pdf[0]`
